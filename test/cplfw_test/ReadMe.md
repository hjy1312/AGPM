Evaluation on CFP----------------------------

How to run:
0.put the aligned data aligned_cplfw/ in this directory;
1.modify the gpu_id and list_name in resnet34_cal_fea_progressive_train_with_pose.py according to your setting;
2.modify the python path and model path in end2end.sh;
3.run progressive_end2end.sh;
4.the evalution results are saved in the directory of the test model path.
